## ✨ 核心功能

- 🤖 **多模型支持** — OpenAI 兼容协议，支持 DeepSeek、Qwen、Ollama 等主流模型
- 🔒 **本地优先** — SQLite 存储，对话历史不上传，数据完全掌控
- 🛠️ **工具调用** — 文件读写、Shell 执行、Web 搜索、代码搜索
- 🔌 **MCP 集成** — Model Context Protocol，扩展工具能力
- 📝 **Git 管理** — 内置 Git 操作（status/diff/commit/push/pull）
- 💻 **终端模拟器** — 内置 PTY 终端，直接执行命令
- 🎨 **现代化 UI** — 暗色主题，项目/会话管理，CodeMirror 编辑器

### 🤔 如何判断我的 Mac 是哪个架构？

点击左上角  → "关于本机"：
- 显示 **Apple M1/M2/M3/M4** → 下载 **Apple Silicon** 版
- 显示 **Intel** → 下载 **Intel** 版
